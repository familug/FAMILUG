## Format code
C-M-\ or M-x indent-region

## Geiser mode
C-c C-z : switch to REPL and back
C-c C-b : eval the buffer (== build and run Scheme code)
C-M-x : eval under cursor
C-c M-p : previous command

## Editting
C-x 1 : single this window
C-x u : undo
C-a C-k: delete line
C-y : paste 
M-w : copy mouse
## Moving other window
C-shift-v : scroll down
C-M-v : scroll up
